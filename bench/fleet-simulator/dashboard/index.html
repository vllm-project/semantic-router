Quick Start
1. Choose a workload — Upload a trace or pick a built-in CDF (Azure, LMSYS, Agent-heavy)
2. Define a fleet — Configure GPU pools and a routing algorithm
3. Run a scenario — Optimize (find minimum GPUs), Simulate (validate a fixed fleet), or What-if (traffic sweep)
4. View results — P99 TTFT, SLO compliance, cost breakdown, utilisation per pool
